## 🚀 Exercícios Implementados

### 1. 🔀 Estruturas Condicionais
- **Arquivo**: `Condicionais.py`
- **Descrição**: Verificação se um número é positivo, negativo ou zero
- **Conceitos**: `if`, `elif`, `else`, entrada de dados

### 2. 🔄 Estruturas de Repetição
- **Arquivo**: `Exercicio_For.py`
- **Descrição**: Soma de números pares em um intervalo
- **Conceitos**: `for`, `range()`, operadores matemáticos

### 3. 🎮 Jogo de Adivinhação
- **Arquivo**: `Jogo_de_adivinhação.py`
- **Descrição**: Jogo interativo onde o usuário deve adivinhar um número
- **Conceitos**: `while`, `random`, tratamento de exceções, lógica de jogo

### 4. 📊 Algoritmos de Busca
- **Pasta**: `Algoritmos_simples/`
- **Descrição**: Implementação de algoritmos fundamentais
- **Conceitos**: busca linear, busca binária, manipulação de listas

### Pré-requisitos
- Python 3.6 ou superior instalado


## 📖 Conceitos por Módulo

### 📚 **Módulo 1: Estruturas de Dados**
- **Listas**: Manipulação, métodos, indexação
- **Tuplas**: Imutabilidade, desempacotamento
- **Dicionários**: Chave-valor, métodos, iteração
- **Sets**: Conjuntos, operações matemáticas

### 🔧 **Módulo 2: Funções** *(Atual)*
- **Funções**: Definição, parâmetros, retorno
- **Escopo**: Variáveis locais e globais
- **Funções Lambda**: Expressões anônimas
- **Decoradores**: Modificação de funções

### 📚 **Módulo 3: Módulos e Bibliotecas**
- **Import**: Importação de módulos
- **Bibliotecas**: Padrão e externas
- **Pacotes**: Organização de código
- **pip**: Gerenciamento de dependências

### 🖥️ **Módulo 4: Interface Gráfica (Flet)**
- **Flet Framework**: Criação de interfaces
- **Componentes**: Botões, campos, layouts
- **Eventos**: Interação do usuário
- **Responsividade**: Adaptação de telas

### 🏗️ **Módulo 5: POO (Programação Orientada a Objetos)**
- **Classes e Objetos**: Conceitos fundamentais
- **Herança**: Reutilização de código
- **Polimorfismo**: Múltiplas formas
- **Encapsulamento**: Proteção de dados
- **Associação**: Relacionamento entre classes

### 🗄️ **Módulo 6: Banco de Dados SQL**
- **SQL Básico**: SELECT, INSERT, UPDATE, DELETE
- **Relacionamentos**: Chaves primárias e estrangeiras
- **Junções**: INNER, LEFT, RIGHT JOIN
- **Python + SQL**: Integração com SQLite/MySQL
